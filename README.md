## Overview

Super Learner Bros is a playable mini-course for AI-curious adults. It turns proven AI learning patterns into a simple world of levels, tricks, and "prompt recipes" they can re-use. The experience feels like a Mario-style world map, but every click helps the user understand new topics faster, break real tasks into steps, evaluate AI tools, and remember more.

## Product Structure

### 4 Worlds, 12 Acts, 12 Prompt Recipes

1. **World 1 â€“ Brain Boss Plains**: Use AI to reduce confusion with multiple explanation styles
2. **World 2 â€“ Dungeon of Hard Tasks**: Break down overwhelming tasks and find the right first step
3. **World 3 â€“ AI Marketplace**: Evaluate AI tools based on real needs, not marketing
4. **World 4 â€“ Memory Mountain**: Use AI to create memorable learning experiences

Each World contains 3 Acts, and each Act teaches a specific AI learning skill through:
- **Read**: Knowledge Tiles explaining the concept
- **Watch**: Show Me Demo of the technique in action
- **Do**: Interactive Mini Game to practice the skill
- **Unlock**: Prompt Recipe saved to "My AI Moves" library

## Target Audience

**Primary Persona: "Jess the Busy Learner"**
- Age 24, junior marketing specialist
- Curious about AI but overwhelmed by tutorials
- Has 20 minutes between tasks to learn
- Wants practical skills for real work (reports, decks, emails)

**Design Principle**: Every screen should make sense in under 10 seconds.

## Key Features

### Learning Components
- **Interactive World Map**: Progress through 4 themed worlds
- **Difficulty Modes**: Chill, Normal, and Boss modes
- **Prompt Library**: Personal collection of unlocked AI techniques
- **Real-World Application**: Use your own tasks and topics, not dummy data

### UX Components
- Top HUD Strip showing skill and progress
- Instruction Banners for mini-games
- Progress Map Indicator
- Difficulty Selector
- Read vs Play Preference Toggle

## Non-Functional Requirements

- **Accessibility**: WCAG AA color contrast, 44px touch targets, alt text
- **Reading Level**: All copy at 8th grade or below
- **Tone**: Friendly, clear, slightly witty
- **Responsiveness**: Desktop-first with mobile considerations
